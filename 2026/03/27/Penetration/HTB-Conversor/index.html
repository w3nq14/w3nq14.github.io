Penetration

Conversor 是一台运行 Flask Web 应用的 Linux 服务器，该应用使用 XSLT 将 nmap XML 输出转换为 HTML。我会找到它的源代码，并利用 os.path.join 的不安全用法，将一个 Python 反向 shell 写入 cron 定时执行的脚本目录；或者，我会滥用 XSLT 的 exslt:document 扩展名，将文件写入服务器。之后，我会找到 SQLite 数据库中经过 MD5 哈希处理的密码，并破解它，从而跳转到下一个用户。对于 root 用户，我会利用 needrestart 中的 CVE-2024-48990 漏洞，通过篡改 PYTHONPATH 环境变量来获取权限；或者，我会滥用 needrestart 的 Perl 配置文件，直接执行代码。

needrestart 是 Ubuntu 系统中常用的一个实用程序，用于检查软件包升级后哪些服务需要重启。它的工作原理是扫描 /proc 目录下的运行进程，以识别那些仍在使用过时共享库的进程。在扫描过程中， needrestart 会执行进程内存映射中找到的解释器二进制文件（例如 Python、Ruby 等），以确定模块依赖关系。

CVE-2024-48990（含 C）

NIST 将 CVE-2024-48990 描述为：

Qualys 发现，在 3.8 版本之前，needrestart 允许本地攻击者通过欺骗 needrestart 运行 Python 解释器并设置攻击者控制的 PYTHONPATH 环境变量，以 root 身份执行任意代码。

当 needrestart 运行 Python 时，它会使用被扫描进程环境中的 PYTHONPATH 变量。这意味着，控制正在运行的进程的攻击者可以设置 PYTHONPATH ，并且有很多方法可以利用这一点来获取执行权限。
